# **App Name**: SyncNote Live

## Core Features:

- Real-time Text Synchronization: Any changes made to the notepad's text by one user are instantly reflected on all other connected devices sharing the same unique link, enabling true live collaboration.
- Unique Live Notepad Links: Generate a unique, shareable URL for each new notepad, allowing users to easily invite collaborators or access their synchronized content from multiple devices.
- Live Editing Interface: A clean and simple text editor interface where users can type, edit, and instantly view all synchronized content across linked devices.
- Content Persistence: Notepad content is automatically saved and remains available persistently when accessed later via its unique shared link, even after all users have disconnected.

## Style Guidelines:

- Primary color: A balanced, inviting blue (#237EC0) for a sense of focus and collaboration, while maintaining a digital and modern feel. HSL (200, 70%, 45%).
- Background color: A very light, desaturated blue (#EBF1F5) providing a clean, paper-like canvas for readability and ease of use. HSL (200, 15%, 95%).
- Accent color: A vibrant teal-green (#1A8989) analogous to the primary color, used sparingly for interactive elements and highlights to add dynamism and contrast. HSL (170, 75%, 40%).
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for both short titles and extended body text in a functional notepad.
- Use minimalist, line-based icons that are intuitive and consistent across the interface, supporting clarity for actions like 'share' or 'new notepad'.
- A clean, single-column layout centered on the text editing area, with a responsive design to ensure seamless functionality on both mobile and desktop devices. The header will be minimal, primarily containing the sharing functionality.
- Subtle visual cues and transitions for real-time updates (e.g., a gentle highlight or fade effect when external changes appear) to enhance the sense of live collaboration without being distracting.
